🔢 How should plant-level data be managed for No Series? — applies to all records in this tab
🏢 Company-wide
All series use a single shared sequence across every plant. Best when your operations are centralised.
e.g. EMP-00001 e.g. TKT-00001

Each plant runs its own independent counter per series. Best when plants operate separately with their own documents.

ℹ️

The Duration (in minutes) field appears only when Reason Type is set to Machine Idle and the Planned checkbox is enabled. The specified duration is treated as In-Process Planned Downtime in OEE monitoring and defines the maximum allowed planned time for this idle reason. If the actual idle time exceeds this defined duration, the excess time is considered as Unplanned Downtime. This configuration impacts downtime reason assignment and OEE calculation by clearly separating planned and unplanned downtime.
